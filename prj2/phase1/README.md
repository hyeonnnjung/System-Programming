# Phase 1

### 컴파일 방법
컴파일을 위해서는 다음 명령어를 터미널에 입력합니다:

`$ make`

#

### 명령어 실행 방법
phase1 에서 지원하는 기본 명령어는 다음과 같습니다:

#### cd (change directory)
디렉토리를 변경합니다:

`$ cd [상대경로/절대경로]`

`$ cd ..`

#

#### mkdir (make directory), rmdir (remove directory)
새로운 폴더를 생성하거나 기존 폴더를 제거합니다:

`$ mkdir [폴더 이름]`

`$ rmdir [폴더 이름]`

#


#### touch, cat, echo
파일을 생성하거나, 파일 내용을 출력하거나, 텍스트를 출력합니다:

`$ touch [파일 이름]`

`$ cat [파일 이름]`

`$ echo [문자열]`

#

### 종료 방법
프로그램을 종료하는 방법은 다음과 같습니다:


`$ exit`


`$ ^C (Ctrl+C)`
