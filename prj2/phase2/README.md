# Phase 2

### 컴파일 방법
컴파일을 위해서는 다음 명령어를 터미널에 입력합니다:

`$ make`

#

### 파이프 명령어 실행 방법
phase2 에서 지원하는 파이프 명령어는 다음과 같습니다:

#### '|' 기호로 명령어 연결하기
ex)

`$ ls | grep [파일 이름]`

`$ cat [파일 이름] | less`

`$ cat [파일 이름] | grep -i "abc" | sort -r`

#


### 종료 방법
프로그램을 종료하는 방법은 다음과 같습니다:


`$ exit`


`$ ^C (Ctrl+C)`
